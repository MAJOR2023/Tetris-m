# Tetris-m

Classic Tetris built with React, TypeScript, and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

| Key | Action |
| --- | --- |
| Arrow Left / Right | Move piece |
| Arrow Down | Soft drop |
| Arrow Up | Rotate |
| Space | Hard drop |
| P | Pause / resume |
| R | Restart |

## Features

- 10×20 playfield with all seven tetrominoes
- Seven-bag randomizer, line clears, scoring, and levels
- Next-piece preview, pause, and game over states
- Best score saved in browser local storage
